This is IOTea, an internet-controlled tea brewer.

This web interface connects to a physical tea-making machine. Simply enter the water temperature and brewing time you want and press the button, and it will heat the water, dispense the boiling water into a tea brewing vessel, brew the tea and dispense it into your cup, all automatically!

This project was made for the 2017 Oxford Hack, and all the code is available on github (link).
